# Target Cards — Rules Handover

Please update the game rules to describe this optional automatic Strain-targeting system. Player Choice remains the default.

## Deck setup

- Use 24 Target Cards. Cards 13–24 repeat the preference combinations on cards 1–12, but each uses the opposite direction arrow.
- Shuffle the full deck once during setup and keep it face down.
- After resolving a draw, return that card face down to the bottom of the deck. There is no discard pile and no reshuffling during play.

## Choosing a Strain target

1. The Burden or effect causing the Strain determines which tiles are eligible. Resolve any printed fallback before drawing.
2. Draw one card for each different target required, including when only one tile is eligible. Multiple Strain placed on the same tile uses one card.
3. Read the card from top to bottom: tile class, tile side, adjacency, then current Strain.
4. At each line, keep the matching candidates if at least one matches. If none match, ignore that line and keep the current candidates. Never return a tile removed by an earlier line.
5. If several candidates remain, choose the tile furthest in the card’s arrow direction using the fixed map orientation. Use fixed map-coordinate order for an exact tie.
6. Return the card to the bottom of the deck, then resolve Supported, Golden Garden, and other prevention. Prevented Strain is not redirected.

Supported tiles remain eligible. Players still choose whether to pay in payment-or-Strain effects; the card determines the Strain target if that branch is used. Use a fresh card for every different target and for Overstrain spread targets.

Do not include the former Priority Die experiment in the rules.
